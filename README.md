# 生成目录脚本

`./source/_node/index.js`



## 项目指令

```bash
# 本地启动项目
hexo server

# 生成文章（把md 文件生成 html 文件）
hexo g

# 上传文章到git（就是发布的意思）
hexo d

```






## 环境需求
