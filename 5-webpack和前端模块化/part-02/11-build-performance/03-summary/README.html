提升构建性能

webpack 性能提升优化分为两类

  1. 利用webpack 提升网站性能，例如网站首屏时间，受益者用户
  2. 开发性能: 提升打包速度，降低打包时间，受益者开发者

三个环境提升构建性能

  1. 通用环境
  2. 开发环境
  3. 生产环境

通用环境优化

  1. 更新到最新版本(webpack, Node.js, npm, yarn等)
  2. 将loader 应用于最少数量的必要模块，比如exclude掉不必要的文件
  3. 引导: 每个额外的loader/plugin 都有其启动时间，尽量少的使用工具
