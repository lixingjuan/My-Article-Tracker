class和hooks对比


|          | hooks                                  | class                                                        |
|----------|----------------------------------------|--------------------------------------------------------------|
| 简洁性   | 简洁                                   | 麻烦                                                         |
| 代码组合 | 逻辑更加聚合                           | 不相关的逻辑出现在同一个生命周期，相同的逻辑放在不同生命周期 |
| 复用性   | 自定义hook实现逻辑服用，数据流向更清晰 | class组件的逻辑复用通常用HOC                                 |
